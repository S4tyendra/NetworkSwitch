<p><i>Network Switch</i> enables you to toggle between 4G and 5G network modes with dual control methods: Root access for rooted devices and Shizuku for non-rooted devices.</p><p><br><b>Purpose</b></p><p>Network Switch allows Android users to manually control their device's network preference between pure 4G (LTE-only) and 5G (NR-only) modes. This is useful for:</p><ul><li>Quick network switching through Quick Settings tile</li><li>Optimizing battery life by forcing 4G mode</li><li>Maximizing speed by forcing 5G mode where available</li><li>Managing data usage and connectivity preferences</li></ul><p>The app provides two methods of operation:</p><ul><li><b>Root Method:</b> Direct system access for rooted devices</li><li><b>Shizuku Method:</b> System access through Shizuku service for non-rooted devices</li></ul><p><b>Features</b></p><ul><li>Pure network mode switching (LTE-only for 4G, NR-only for 5G)</li><li>Quick Settings tile for instant access</li><li>Dual control methods (Root and Shizuku)</li><li>Modern Material Design 3 interface</li><li>Clean architecture implementation</li><li>Privacy-focused (no internet permissions)</li><li>Intelligent device compatibility detection</li><li>Automatic fallback for unsupported devices</li></ul>
